📢 ประกาศถึงผู้เช่าทั้งหมด

ส่งข้อความหาผู้เช่าปัจจุบันทุกคนที่ลงทะเบียน LINE แล้ว

ข้อความจะถูกส่งหาผู้เช่าที่มีสถานะเช่าอยู่ปัจจุบันเท่านั้น

💧 ผู้มีสิทธิ์บันทึกน้ำดื่มผ่าน LINE

เฉพาะ LINE User ID ที่ระบุเท่านั้นที่สามารถพิมพ์คำสั่งบันทึกน้ำดื่มใน LINE OA ได้
ถ้าไม่ระบุ = ทุกคนที่ไม่ใช่ผู้เช่าบันทึกได้

ไม่รู้ LINE User ID? ส่งข้อความอะไรก็ได้หา LINE OA แล้ว Bot จะ reply ID กลับมา
